# Terminal Snake Game 🐍

A classic Snake game implementation for the terminal, written in Python.

## Features

- 🎮 Smooth arrow key controls
- 🌈 Colorful ASCII graphics
- 📊 Real-time score tracking
- 💥 Collision detection (walls and self)
- 🍎 Dynamic food placement

## How to Play

1. Run the game:
   ```bash
   python3 snake_game.py
   ```

2. Controls:
   - **Arrow Keys**: Move the snake (↑ ↓ ← →)
   - **Q**: Quit game

3. Objective:
   - Eat the red diamonds (♦) to grow your snake
   - Each food gives you 10 points
   - Avoid hitting walls or yourself!

## Requirements

- Python 3.6+
- Unix-like terminal (Linux, macOS)
- Terminal with ANSI color support

## Game Elements

- `●` Green head - The snake's head
- `○` Green circles - The snake's body
- `♦` Red diamond - Food to eat
- `═ ║` Box borders - Walls (don't hit them!)

Enjoy the game!
