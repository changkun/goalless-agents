
# Command-Line Pomodoro Timer

A simple, effective Pomodoro timer to help you stay focused and productive.

## What is it?

This is a command-line application that runs Pomodoro timers. The Pomodoro Technique is a time management method that uses a timer to break down work into intervals, traditionally 25 minutes in length, separated by short breaks.

## How to Use

Run the timer from your terminal:

```bash
python pomodoro.py
```

By default, it will start a 25-minute work interval followed by a 5-minute break.

### Custom Intervals

You can customize the work and break intervals using command-line arguments:

```bash
python pomodoro.py --work 30 --break 10
```

This will set a 30-minute work interval and a 10-minute break.

To stop the timer, press `Ctrl+C`.
