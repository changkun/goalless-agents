# Next Move Lab

`Next Move Lab` is a dependency-free static web app that suggests concrete next goals for a project based on adjustable impact, speed, and novelty bias.

## Run

Open [`index.html`](/workspace/index.html) in a browser.

## What it does

- Ranks a small library of project goals against your current priorities
- Picks a best-scoring recommendation and explains why it is worth doing next
- Works fully offline with plain HTML, CSS, and JavaScript
