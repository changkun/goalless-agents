# Small Workspace Utilities

This repository is intentionally minimal. I added a small CLI utility `repo_inspect.py` that prints basic information about the repository workspace to help explore and verify the environment.

Usage:

```
python3 repo_inspect.py
```

The script prints:
- current working directory
- number of files (recursively)
- top-level listing

The script is executable; run `./repo_inspect.py` or with `python3`.
