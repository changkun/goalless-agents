# Terminal Snake Game 🐍

A classic Snake game built for the terminal using Python and curses.

## Features

- Smooth snake movement with arrow key controls
- Score tracking
- Clean terminal UI with box-drawing characters
- Game over detection with restart option
- Food spawning system
- Collision detection

## How to Play

Run the game:
```bash
python3 snake.py
```

### Controls
- **Arrow Keys**: Move the snake (Up, Down, Left, Right)
- **Q**: Quit the game
- **R**: Restart after game over

### Rules
- Eat the food (◆) to grow longer and gain points
- Don't hit the walls or yourself
- Each food item gives you 10 points

## Requirements

- Python 3.x
- Terminal with curses support (built-in on Linux/macOS)

Enjoy the game!
